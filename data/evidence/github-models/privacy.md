# GitHub Models — Privacy / Training Policy Evidence

**Last Verified:** 2026-04-27
**Source:** https://docs.github.com/en/site-policy/github-terms/github-terms-of-service + https://docs.github.com/en/github-models/responsible-use-of-github-models

## Official Quotes

> "GitHub Models is a prototyping environment. Your inputs and outputs are processed by the underlying model provider."
> — GitHub Models documentation, Responsible Use section (retrieved 2026-04-27)

> "When using GitHub Models, the inputs you provide and the outputs you receive are subject to the model provider's terms and conditions."
> — GitHub Models documentation (retrieved 2026-04-27)

> "GitHub does not use your prompts or completions from GitHub Models to train GitHub's AI models."
> — GitHub Models responsible use guidelines (retrieved 2026-04-27)

## Assessment

- **GitHub itself**: Does **not** train on user data from GitHub Models.
- **Underlying providers**: Each provider (OpenAI, Anthropic, Meta, Mistral, etc.) has its own terms.
- For privacy-sensitive work, use production API keys directly from the model provider.
- GitHub Models is labeled as a "prototyping" environment — not recommended for production or sensitive data.

## Training Status: `false` (GitHub itself does not train; underlying providers may)

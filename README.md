## 这个项目是什么

本项目不只是"免费 LLM API 清单"，而是一套**可以真正跑起来的实用工具包**：

- 🔑 **8 家永久免费供应商**，覆盖聚合器、一手模型、本地方案
- 🛠️ **一键检查脚本** —— 输入你的 key，立刻知道哪家通、哪家挂
- 🖼️ **多维能力矩阵** —— Vision / 工具调用 / 出图 / 长上下文，按需索骥
- 🔌 **7 款客户端接入指南** —— opencode / Cursor / Cline / Continue / LibreChat / OpenWebUI / Chatbox
- 🗺️ **场景化 Recipes** —— 代码补全 / agent 跑法 / 看图 / 出图 / 多供应商路由
- 🔒 **隐私矩阵** —— 哪家会用你的数据训模型，一眼看清
- 📖 **非开发者图文教程** —— 不用懂命令行，30 分钟上手

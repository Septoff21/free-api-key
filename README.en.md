## What is this project?

Not just another "free LLM API list" — this is a **working toolkit**:

- 🔑 **8 permanently-free providers** — aggregators, first-party, and local options
- 🛠️ **One-command key checker** — paste your keys, instantly see which work
- 🖼️ **Multi-dimensional capability matrix** — Vision / Tool-use / Image-gen / Long-context
- 🔌 **7 client integration guides** — opencode / Cursor / Cline / Continue / LibreChat / OpenWebUI / Chatbox
- 🗺️ **Scenario-based Recipes** — code completion, agent workflows, vision, image gen, multi-provider routing
- 🔒 **Privacy matrix** — which providers train on your data, at a glance
- 📖 **Non-developer illustrated tutorials** — no terminal required, up and running in 30 min

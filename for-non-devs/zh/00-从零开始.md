## 第一步：选一家 AI 提供商

**AI 提供商** = 提供 AI 能力的公司。你注册后会拿到一串密码（叫 "API Key"），把它填进聊天软件里就能用。

根据你的情况选择：

| 我的情况 | 推荐选择 | 难度 |
|---|---|:-:|
| 想最快用上，有谷歌账号 | **Gemini**（谷歌出品）| ⭐ 最简单 |
| 想用更多 AI 模型 | **OpenRouter** | ⭐⭐ 简单 |
| 完全不想联网 / 非常注重隐私 | **Ollama**（本地运行）| ⭐⭐⭐ 需要安装软件 |

**新手推荐：先选 Gemini 或 OpenRouter，用 Chatbox 作为聊天软件。**

---

## 第二步：拿到 API Key

按你选择的提供商，看对应的图文教程：

- 📖 [OpenRouter 图文教程](01-openrouter图文.md)（推荐：模型多）
- 📖 [Gemini 图文教程](02-gemini-tuweng.md)（推荐：最简单）

拿到 Key 之后，它长这个样子（不同提供商格式不同）：

```
OpenRouter: sk-or-v1-xxxxxxxxxxxxxxxxxxxxxxxx
Gemini:     AIzaSyXXXXXXXXXXXXXXXXXXXXXXXX
```

> ⚠️ **Key 是你的"密码"** — 不要发给别人，不要截图后上传到网上。

## 第三步：安装 Chatbox

Chatbox 是一款免费、中文界面的 AI 聊天软件，支持 Windows / macOS / Linux / 手机。

1. 打开浏览器，访问 [https://chatboxai.app](https://chatboxai.app)
2. 点击"免费下载" 
3. 按你的系统选择安装包
4. 安装完毕，打开软件

[截图占位 - chatbox-01-homepage.png：Chatbox 官网首页，箭头指向下载按钮]

---

## 第四步：把 API Key 填进 Chatbox

### 如果你用的是 Gemini

1. 打开 Chatbox → 左下角点击 **设置（齿轮图标）**
2. 找到 **AI 提供商** → 选择 `Google Gemini`
3. **API Key** 栏粘贴你从 Gemini 拿到的 Key（`AIzaSy...`）
4. **模型** 选 `gemini-2.0-flash`
5. 点击 **保存**

[截图占位 - chatbox-02-gemini-setup.png：Chatbox 设置界面，填写 Gemini Key]

### 如果你用的是 OpenRouter

1. 打开 Chatbox → 左下角 **设置**
2. **AI 提供商** → 选 `OpenAI API`（注意：不是 "Chatbox AI"）
3. **API Host** 填写：`https://openrouter.ai/api/v1`
4. **API Key** 粘贴你的 OpenRouter Key（`sk-or-v1-...`）
5. **模型** 手动输入：`deepseek/deepseek-chat:free`
6. 点击 **保存**

[截图占位 - chatbox-03-openrouter-setup.png：Chatbox 设置界面，填写 OpenRouter 信息]

---

## 第五步：开始聊天！

1. 关闭设置界面，回到主聊天界面
2. 点击 **+ 新建对话**
3. 在底部输入框输入你的第一条消息，按回车发送

**建议的第一条消息：**
```
你好！请用中文介绍一下你自己，你能帮我做什么？
```

正常情况下 5 秒内会收到回复。

[截图占位 - chatbox-04-first-chat.png：Chatbox 聊天界面，AI 正在回复]

🎉 **恭喜！你现在拥有了一个免费的 AI 助手。**

## 常见问题 FAQ

### ❓ 提示 "401 Unauthorized"（401 错误）

**原因：** API Key 填错了，或者多了空格。  
**解决：** 回到设置界面，把 Key 删掉，重新完整地复制粘贴一遍。

### ❓ 提示 "429 Too Many Requests"（429 错误）

**原因：** 用得太频繁，触发了免费层的速率限制。  
**解决：** 等 1 分钟后再试。这是正常的，免费层有每分钟请求数限制。

### ❓ 转圈很久没有回复

**原因：** 可能是网络问题（某些 AI 服务在中国大陆需要代理访问）。  
**解决方案：**
  - 换用 Gemini（`generativelanguage.googleapis.com` 在大部分地区可直连）
  - 如果有代理工具，开启代理后再试
  - 切换到本地 Ollama（完全不联网）

### ❓ 模型选项是空的

**原因：** 客户端版本太旧，或者 API Host 填错了。  
**解决：** 更新 Chatbox 到最新版本；检查 API Host 是否带了 `https://`。

### ❓ Gemini 说"你的数据可能被用于训练"

**说明：** Gemini 免费层确实可能使用对话数据训练模型（谷歌官方政策）。  
**如果你介意：** 改用 OpenRouter + DeepSeek（OpenRouter 声明不训练数据），或用本地 Ollama。

### ❓ 我不小心把 Key 分享出去了怎么办

1. 立刻去对应提供商的网站，找到 "API Keys" 管理页面
2. 点击 "Revoke"（吊销）这个 Key
3. 重新生成一个新 Key
4. 在 Chatbox 里更新新 Key
